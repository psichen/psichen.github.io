糖类摄入与皮肤衰老

Posted on 2018-04-09

简单研究了一下糖类摄入和皮肤衰老之间的关系，如有错误请指正~

什么是糖类

糖类主要由碳、氢、氧三种元素组成，早期发现的糖类分子符合Cm(H2O)n的结构式，因此称之为碳水化合物，不过后来发现有例外，如鼠李糖(rhamnose, C6H12O5)。

糖类又分为单糖、双糖和多糖，单糖如葡萄糖、果糖和半乳糖，双糖如乳糖，多糖如淀粉和纤维素。人体将多糖或双糖大部分分解为单糖后，经过肠道吸收进入血液并运输到全身细胞当中，在细胞被降解为丙酮酸，并在线粒体里酶的催化下进一步降解为乙酰辅酶A，进入三羧酸循环，与氧气发生一系列的反应，最终代谢为二氧化碳和水，其中产生的能量以ATP的形式给细胞供能。

人体可以将多余的血糖以多糖或脂肪的形式存储起来，其中人体的多糖称之为糖原，分为肝糖原和肌糖原。肝糖原里的葡萄糖可以调节血糖水平，肌糖原里的葡萄糖就近给肌肉供能。而糖原的存储量毕竟有限，当仍存在多余的血糖时，人体就会将其转化为脂肪细胞。

并不是只有吃起来是甜的东西才是糖类，很多东西虽然不甜，但糖分含量非常高，如米饭、土豆、玉米和面条等，并且这些食物的GI(glycemic index, 升糖指数)非常高，吃下后短时间内会让血糖水平急剧增加，此时身体会开始分泌胰岛素，下调血糖水平，使人饿得更快，同时多余的血糖也会转化为脂肪存储起来。

胶原蛋白(collagen)

胶原蛋白相信大家已经耳熟能详了，胶原蛋白是最结实的蛋白之一。在皮肤里胶原蛋不仅为组织和细胞起到支撑的作用，还会和细胞相互作用，影响细胞迁移、分化和增殖等过程。

在空间结构上，胶原蛋白显示出特殊的三股螺旋缠绕的结构，三条相互独立的胶原蛋白肽链依靠甘氨酸之间形成的氢键维系三股螺旋相互缠绕的结构。胶原蛋白这种特殊的三股螺旋结构保证了它的机械强度。这种三股螺旋被称为原胶原。

若干个原胶原横向堆积，序列中所含有的羟赖氨酸和羟脯氨酸侧链在酶作用下氧化生成醛，相互之间发生羟醛缩合反应形成原胶原之间的共价链接，这种结构被称为胶原微纤维。许多胶原微纤维横向堆积，以相同的方式通过共价键链接，形成胶原纤维。胶原纤维是胶原蛋白行使生理作用的基本形态，在生物体内胶原纤维交织成富有机械强度和弹性的网状结构成为结缔组织最基本的组成成分。 - Wikipedia

从上图可以看到，胶原纤维在真皮层中起到支撑作用，使得表皮光滑无皱纹，而胶原纤维减少或失去弹性后，表皮层就会塌陷，使人看上去衰老。

糖基化(glycation)

衰老的机制一直是一个经久不衰的课题，而对于皮肤衰老来说，近年来晚期糖基化终末产物(advanced glycation end products, AGEs)越来越受到关注。

何为糖基化？还原糖如单糖分子结构上含有一个醛基，可与蛋白上赖氨酸或精氨酸上的氨基反应，发生缩合，这个反应就称之为糖基化反应，最终生成的产物就称之为AGEs。

通常在20岁的时候就可以观察到胶原蛋白发生糖基化，并以每年平均3.7%的幅度积累，最终在80岁的时候达到30-50%的水平。

胶原蛋白发生糖基化后会在不同方面影响其功能，促使皮肤衰老：

  1. 分子间的胶原纤维的交联会使其机械性质发生改变，变得更加僵硬而丧失弹性。
  2. 糖基化后胶原纤维的电荷会发生改变，同时侧链上会形成AGEs，这都会影响胶原纤维和细胞及其他基质蛋白的接触，从而抑制其相互作用，胶原蛋白单体形成三聚体的过程也会受到影响。
  3. 糖基化后的胶原蛋白难以被降解，使得新生成的胶原蛋白难以替换掉原有的胶原蛋白。
  4. AGEs会结合细胞表面受体RAGE，传导细胞信号，减少细胞增殖，诱导细胞凋亡等。
  5. AGEs可导致ROS（活性氧）的产生，可产生超氧化阴离子，破坏细胞的抗氧化系统。

抗糖化tips

  1. 防晒
    在年轻个体中，在对紫外保护较好的皮肤部位里没有发现AGEs的积累，而主要在暴露于阳光的皮肤部位发现了AGEs的积累。表明紫外对皮肤的伤害除了促炎、细胞凋亡、氧化损伤以及基因突变等，还会促使AGEs的形成。

  2. 戒烟
    抽烟有百害而无一利。

  3. 控制糖类摄入量
    戒掉可乐、蛋糕等高含糖量食物，控制高GI食物的摄入，多摄入低GI食物，也会更加经饿。另外预防糖尿病也非常重要。

  4. 抗氧化
    氧化损伤也是皮肤衰老的一大因素，在糖基化过程中少不了氧气的参与，如果通过外敷或内服的方式，使得抗氧化剂消耗掉氧气，那么也可以延缓皮肤糖基化的过程，同时也能减缓皮肤的氧化损伤。
