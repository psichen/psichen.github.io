Permutation

Let’s first consider a set of nnn objects, which are all different. The number of all possible arrangements (permutations) is

n(n−1)⋯1=n!n(n-1) \cdots 1 = n! n(n−1)⋯1=n!

Generally, if we select k(<n)k (<n)k(<n) objects from nnn, the number of permuations is

n(n−1)⋅⋅⋅(n−k+1)=n!(n−k)!=P(n,k)n(n-1)\cdot\cdot\cdot(n-k+1) = \frac{n!}{(n-k)!} = P(n, k) n(n−1)⋅⋅⋅(n−k+1)=(n−k)!n!​=P(n,k)

Previously we assumed objects were selected without replacement. If the selected object is put back into the population every time, it’s called sampling with replacement. Easily, If kkk objects are selected from nnn objects with replacement, the number of permutations is

Combination

We now consider the number of combinations of various objects when their order is immaterial. We have already known that the number of permuations of kkk objects from nnn is P(n,k)=n!(n−k)!P(n, k)=\frac{n!}{(n-k)!}P(n,k)=(n−k)!n!​, of which the permuations are repeated k!k!k! times. So the number of distinguishable arrangements (combinations) of kkk objects from nnn is

n!k!(n−k)!=C(n,k)\frac{n!}{k! (n-k)!} = C(n, k) k!(n−k)!n!​=C(n,k)

Let’s consider nnn objects of which n1n_1n1​ are identical and of type1, n2n_2n2​ are identical and of type2, and nmn_mnm​ are identical and of type mmm. The permutation of these nnn objects is n!n!n!. However, the number of distinguishable arrangements is

n!n1!n2!⋯nm!\frac{n!}{n_1! n_2! \cdots n_m!} n1​!n2​!⋯nm​!n!​

It’s also called multinomial coefficients in polynomial equation (x1+x2+⋯+xm)n(x_1+x_2+ \cdots +x_m)^n(x1​+x2​+⋯+xm​)n, where nnn represents the total power number of the equation, and n1,n2,⋯nmn_1, n_2, \cdots n_mn1​,n2​,⋯nm​ represents the power number of different variables in each expansion factors.

with replacement

As in the case of permutations we might ask how many combinations of kkk objects can be chosen from nnn objects with replacement. To calculate the value, we may image that the nnn distinguishable objects set out on a table. Each combination of kkk objects can then be made by pointing to kkk objects of the nnn objects in turn. There kkk equivalent selections amongst nnn different but re-choosable objects are strictly analogous to the placing of kkk indistinguishable balls into nnn different boxes with no restriction on the number of balls in each box.

Four different kinds of particle statistics

Let’s consider a partical scenario. A system contains a number NNN of non-interacting particles, each of which can be in any of quantum states of the system. The structure of the set of quantum states is such that there exist RRR energy levels with corresponding energies EiE_iEi​ and degeneracies gig_igi​ (i.e. the iiith energy level contains gig_igi​ quantum states). Find the numbers of distinct ways in which the particles can be distributed among the quantum states of the system such that the iiith energy level contains nin_ini​ particles, for i=1,2,⋯ ,Ri=1,2,\cdots,Ri=1,2,⋯,R, in the cases where the particles are

  1. distinguishable with no restriction on the number in each state;
  2. indistinguishable with no restriction on the number in each state;
  3. indistinguishable with a maximum of one particle in each state;
  4. distinguishable with a maximum of one particle in each state;

Distinguishable w/o amount restriction

Let us first consider distributing the NNN patricles among the RRR energy levels, which is analogous to placing NNN different balls into RRR boxes,

N!ΠRi=1ni!\frac{N!}{\underset{i=1}{\overset{R}{\large{\Pi}}} n_i!} i=1ΠR​ni​!N!​

where nin_ini​ is the number of particles in each energy level. Then let’s multiply the number of differnet arrangement of nin_ini​ particles in gig_igi​ degenerate quantum states in each energy level. Becasue there is no restriction on the particle number in each quantum states, each particle can reside in any of gig_igi​ degenerated quantum states. So the arrangements in this scenario is

Ω=N!ΠRi=1ginini!\Omega = N! \underset{i=1}{\overset{R}{\LARGE{\Pi}}} \frac{g_i^{n_i}}{n_i!} Ω=N!i=1ΠR​ni​!gini​​​

Such a system of particles (for example classical gas molecules) is said to obey Maxwell-Boltzmann statistics.

Indistinguishable w/o amount restriction

If the particles are indistinguishable, there is only one arrangement in which ni(i=1,2,⋯ ,R)n_i (i=1,2,\cdots,R)ni​(i=1,2,⋯,R) particles reside in iiith energy level. The distinct arrangements of nin_ini​ particles in gig_igi​ degenerate states, however, is analogous to placing indistinguishable balls into different boxes. So the number of combination of particles is

Ω=ΠRi=1(ni+gi−1)!ni!(gi−1)!\Omega = \underset{i=1}{\overset{R}{\LARGE{\Pi}}} \frac{(n_i+g_i-1)!}{n_i! (g_i-1)!} Ω=i=1ΠR​ni​!(gi​−1)!(ni​+gi​−1)!​

Such a system of particles (for example a gas of photons) is said to obey Bose-Einstein statistics.

Indistinguishable w/ amount restriction

If the number of particles in each quantum state is either 0 or 1, the arrangements of nin_ini​ particles in gig_igi​ degenerate quantum states is equal to the combination of nin_ini​ occupied and gi−nig_i-n_igi​−ni​ unoccupied states. So the distinct arrangements of particles is given by

Ω=ΠRi=1gi!ni!(gi−ni)!\Omega = \underset{i=1}{\overset{R}{\LARGE{\Pi}}} \frac{g_i!}{n_i!(g_i-n_i)!} Ω=i=1ΠR​ni​!(gi​−ni​)!gi​!​

Such a system of particles (for example an electron gas) is said to obey Fermi-Dirac statistics.

Distinguishable w/ amount restriction

If the particles are distinguishable, each arrangement of nin_ini​ particles in the previous scenario can be reordered in ni!n_i!ni​! different ways. Multiplying the number of arrangements of NNN distinguishable particles among RRR energy levels, the arrangement of particles is given by

Ω=N!ΠRi=1gi!ni!(gi−ni)!\Omega = N! \underset{i=1}{\overset{R}{\LARGE{\Pi}}} \frac{g_i!}{n_i! (g_i-n_i)!} Ω=N!i=1ΠR​ni​!(gi​−ni​)!gi​!​

Such a system of particles never occurs in nature.
